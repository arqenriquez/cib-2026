# Imágenes (assets)

Coloca aquí estos archivos con **exactamente** estos nombres:

| Archivo | Uso | Recomendación |
|---|---|---|
| `abp-logo.png` | Logo ABP en header y footer | PNG con fondo transparente, ~400×400 px |
| `hero-bg.jpg` | Fondo del hero y del bloque de registro | Foto de Biblia/luz, horizontal, ~1920×1280 px, < 400 KB |
| `flyer.jpg` | Imagen al compartir en WhatsApp/redes (Open Graph) | El flyer oficial, ~1000×1600 px |

Si no colocas `hero-bg.jpg`, el hero usa un degradado crema de respaldo (se ve bien igual).

> Optimiza las fotos (TinyPNG / Squoosh) antes de subirlas para que la página cargue rápido en celular.
